# 64-bit 位元運算與欄位提取工具 (Bit Field Extractor)

這是一個輕量、直覺的網頁工具，專為底層開發者、嵌入式工程師或學習計算機結構的人員設計。它可以幫助你快速進行進位轉換、位移運算，並精確提取特定的 Bit Field。

## 🚀 線上使用
https://jk-yys.github.io/bit-field-tool/

## ✨ 功能亮點
- **多進位顯示**：同步顯示 10 進位、16 進位及 2 進位。
- **值域切換**：支援 32-bit 與 64-bit 無符號整數 (Unsigned) 運算。
- **視覺化位移**：提供左移 (<<) 與右移 (>>) 操作，並即時對比移動前後的位元變化。
- **精確欄位提取 (Bit Field Extraction)**：
  - 自定義提取範圍 (High Bit ~ Low Bit)。
  - **自動分組**：提取結果每 4-bits 自動間隔，並對齊對應的 16 進位字元。
  - **精確顯示**：提取幾個 Bit 就顯示幾個，不強制補零，排版更清晰。

## 🛠 如何在本地執行
1. 下載 `index.html`。
2. 使用任何現代瀏覽器（Chrome, Edge, Firefox）開啟即可使用。

## 📸 介面預覽
- **輸入區**：支援 `0x` 前綴輸入。
- **對齊顯示**：2 進位上方標註 Hex 字元，方便口算轉換。
- **綠框區域**：專屬的提取結果顯示區。
